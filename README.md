# CarePlus Multispeciality Clinic

A premium static clinic website built with Next.js and Tailwind CSS for CarePlus Multispeciality Clinic in Kanpur.

## Stack
- Next.js
- React
- Tailwind CSS
- Vercel-ready static deployment

## Features
- Responsive medical landing page
- About, Services, Doctors, Appointments, and Contact pages
- No database, no login, no admin panel
- Premium clinic-style imagery and presentation

## Local Development
```bash
npm install
npm run dev
```

## Production Build
```bash
npm run build
npm run start
```

## Notes
- Appointment form is static in this trial build.
- Contact and appointment flows can later be wired to email, WhatsApp, or Sheets without adding a database.
